# em draft start

### Purpose

Create a new draft from the current head revision.

### Syntax

```bash
em draft start -n "<message>"
```

### Expected `data`

```json
{
  "draft": {
    "id": "draft_008",
    "baseRevisionId": "rev_003",
    "status": "open",
    "message": "Add refund status to payment detail"
  }
}
```

### Example

```bash
em draft start -n "Add refund status to payment detail"
```
