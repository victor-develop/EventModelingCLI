# em project init

### Purpose

Create a new `ModelingProject`.

### Syntax

```bash
em project init <name>
```

### Expected `data`

```json
{
  "project": {
    "id": "proj_payments",
    "name": "Payments Modeling",
    "headRevisionId": null
  }
}
```

### Example

```bash
em project init "Payments Modeling"
```

```json
{
  "ok": true,
  "command": "em project init",
  "projectId": "proj_payments",
  "data": {
    "project": {
      "id": "proj_payments",
      "name": "Payments Modeling",
      "headRevisionId": null
    }
  },
  "warnings": []
}
```
