# em project open

### Purpose

Open an existing project and make it the current context.

### Syntax

```bash
em project open <id|name>
```

### Expected `data`

```json
{
  "project": {
    "id": "proj_payments",
    "name": "Payments Modeling",
    "headRevisionId": "rev_003"
  },
  "currentRevision": {
    "id": "rev_003",
    "message": "Add refund timeline"
  }
}
```

### Example

```bash
em project open proj_payments
```

```json
{
  "ok": true,
  "command": "em project open",
  "projectId": "proj_payments",
  "revisionId": "rev_003",
  "data": {
    "project": {
      "id": "proj_payments",
      "name": "Payments Modeling",
      "headRevisionId": "rev_003"
    },
    "currentRevision": {
      "id": "rev_003",
      "message": "Add refund timeline"
    }
  },
  "warnings": []
}
```
